Hexo next主题中添加播放器Aplayer

​ 在搭建博客过程中，音乐是必不可缺的，就我了解而言，在Hexo中插入音乐有两种方式：1.网易云外链，由于很多歌曲版权限制我们不能生成外链，所以这种不行。2.Aplayer插件， 首先，先下载Aplayer源码：APlayer ， 再将文件中的dist文件夹复制到路径：/themes/next/source 在目录/themes/next/source/dist下添加music.js文件

const ap = new APlayer({
container: document.getElementById(‘aplayer’),
fixed: true,
autoplay: true,
audio: [
{
name: ‘世间美好与你环环相扣’,
artist: ‘柏松’,
url: ‘http://music.163.com/song/media/outer/url?id=1363948882.mp3',
cover: ‘/images/3.jpg’,
},
]
});

注：里面的代码可以修改，歌曲信息需要手动添加：

  • name：歌曲名称
  • artist：作者
  • url：连接（具体怎样操作看下面步骤）
  • cover：是图片连接，可以是本地图片，也可以是在线图片，本地放在source/images然后修改路径即可

至此一个大问题出现了，aplayer插件间断播放

在我们页面变化时，aplayer就会重新加载，这搁谁都不能接受。于是我就去网上查到了：

博主原文

其实在主题配置文件里打开pjax，就行了,在最开始进入网站的时候加载，之后都不会重新刷新页面，播放器就不会断了！
